Project Overview

Starquo invites qualified service providers to deliver a secure, mobile-first platform for an established news publisher.

Primary Goals
  • Migration of framework and verification of all backup content (text, images, video)
  • Full hosting independence and ownership of source code
  • Delivery of a modern, SEO-friendly responsive website
  • Enhanced user experience across all devices
Innovation & Technology
  • AI-powered customer support tools
  • Automated backend workflows
  • Integrated e-paper provider and e-reader viewer
  • Real-time analytics dashboard

Technical Requirements

Hosting Environment
  • Quad Core CPU or higher
  • 16 GB RAM / 256 GB SSD minimum
  • Scalable server with full root access
  • cPanel or equivalent control panel
  • Daily automated backups
CMS & Mobile
  • WordPress or Drupal CMS
  • Hybrid framework (e.g., Flutter)
  • iOS and Android Apps synced with CMS
  • Push notification capabilities
  • Offline content access
Security & Performance

Enterprise-grade SSL certificate, DDoS protection, Web Application Firewall (WAF), load balancing, daily/weekly backups, and Lighthouse performance compliance (90+ scores).

Features & Integrations

100% Code Ownership
24/7 Support
99.9% Uptime SLA
30+ Integrations
  • Revenue Systems: Google AdSense, newsletter automation, subscription management, and paywall support with multiple payment gateways.
  • Analytics: Google Analytics 4, Search Console integration, and custom dashboard.
  • Strategic Partnerships: Integration with Desert FM and Whatson platforms.
  • Assets: Full ownership of database, source code, and hosting infrastructure by Starquo.
  • Content Delivery: CDN integration for global performance optimization.

  • 6 months post-launch support
